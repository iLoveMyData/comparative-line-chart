# Support — Comparative Line Chart

_Last updated: 22 April 2026_

Thank you for using **Comparative Line Chart**. This page explains how to
report a problem, ask a question or request a feature.

## Before contacting support

1. Make sure you are running **Power BI Desktop December 2023** (API 5.11.0)
   or later.
2. Check that the visual has all three required fields bound — **Period**,
   **Series A** and **Series B**.
3. Try reproducing the issue in a fresh `.pbix` using the sample dataset
   provided in the repository (`test/sales-vs-budget.pq` or
   `test/sales-vs-budget-2024-2025.csv`).

## How to get help

### Bug reports and feature requests

Open an issue on the GitHub tracker — this is the preferred channel:

**https://github.com/iLoveMyData/comparative-line-chart/issues/new**

Please include:

- a short description of the problem and the expected behaviour;
- the Power BI Desktop version (Help → About) or the Power BI Service tenant;
- a minimal `.pbix` that reproduces the issue **or** a screenshot and the field
  bindings you used;
- for rendering bugs: browser (if on Service), screen resolution and the
  visual's size in pixels.

### Email

For everything that does not fit an issue (private data, commercial enquiries,
certification contacts):

**karar.sammy@outlook.fr**

We aim to answer within **5 business days**.
